# csv2png

ブラウザで動く軽量な CSV → PNG テーブル変換アプリです。

## 使い方

1. `index.html` をブラウザで開く（またはローカルサーバー配信）。
2. CSVファイルを選択、またはCSVテキストを貼り付ける。
3. アスペクト比 / DPI / フォント / タイトル行列 / デザイン / 色テーマを調整。
4. 「プレビュー更新」で確認し、「PNGをダウンロード」で保存。

## 主な要件対応

- 入力: CSVファイル・テキスト入力の両対応
- 出力: PNG
- 調整項目:
  - アスペクト比（デフォルト16:9）
  - DPI（デフォルト180）
  - フォント（デフォルト Noto Sans JP）
  - タイトル行あり/なし（デフォルトあり）
  - タイトル列あり/なし（デフォルトなし）
  - 表デザイン 5種類（シンプル3・装飾2）
  - カラーパターン 5種類（グレー2・青2・緑1）
